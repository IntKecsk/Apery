Apery

A Penrose tiling generator and cellular automata simulator

View On GitHub
Downloads:
  • ZIP
  • TAR

    Apery is an application for Penrose tiling generation and simulation of cellular automata on them.

    Implemented:

    • Penrose rhomb tiling representation
    • Penrose tiling generation using triple deflation
    • Basic Penrose tiling rendering
